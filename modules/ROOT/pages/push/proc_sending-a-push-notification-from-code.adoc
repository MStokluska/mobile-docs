= Sending a Push Notification from Code

The {push-ui} allows you to send {push-notification}s to devices from code.

WARNING: Due to security reasons, do not store the *Application ID* and the *Master Secret* on a mobile device.

[discrete]
== Prerequisites

* Access to the {push-ui}.
* An application with at least one binding or variant.

[discrete]
== Procedure

. Open the {push-ui} in a browser.
