= Understanding Activity Logs

{unifiedpush-service} collects metadata and payload, and this is accessed using {unifiedpush-service} Activity Logs screen.

IMPORTANT: {unifiedpush-service} *automatically* deletes the collected metadata and payload after 30 days by an out-of-hours batch job.
